# Cubbly iOS — v3

## To run

This project is generated by **XcodeGen**. The included `project.yml` is the
source of truth.

### One-time setup

```bash
brew install xcodegen
```

### Generate the Xcode project

From the `ios-native/` folder:

```bash
xcodegen generate
open Cubbly.xcodeproj
```

Then in Xcode: select the `Cubbly` target → Signing & Capabilities → set your
Development Team → Run.

### What's bundled (verified)

- `Resources/Assets.xcassets/` — AppIcon (cocoa background + bear), brand image
  sets (`cubbly-logo`, `cubbly-nobg`, `cubbly-wordmark`), `LaunchBackground`
  color, `AccentColor`.
- `Resources/LaunchScreen.storyboard` — cocoa background + centered bear logo.
- `Resources/Fonts/` — 7 Nunito + 7 italic static weights (no variable file).
- `Resources/Icons/` — all 28 brand SVGs.
- `Resources/Images/` — brand PNGs (also mirrored in the asset catalog).
- `Resources/Videos/cubbly-loading.mov` — splash animation.

All resource folders are listed as **folder references** in `project.yml` so
they survive opening the .xcodeproj before re-running xcodegen.

## Fixes in v3

- **Resources actually bundle**: folder refs + asset catalog migration.
- **Fonts**: switched to 14 static Nunito faces — kills every "Unable to update
  Font Descriptor's weight" warning. New `Font.cubbly(size, weight)` helper.
- **Launch**: real `LaunchScreen.storyboard` (cocoa + bear), then in-app .mov
  splash animation continues.
- **Navigation**: removed custom HorizontalSwipe gesture entirely — chat now
  uses the native iOS edge-swipe-to-pop, so vertical scroll works again, no
  black flash, no "right-swipe on DM list" bug.
- **Presence**: subscribe order fixed to register callbacks before
  `subscribe()`, then `track()` after.
- **You-tab banner**: `AnimatedImageView` no longer reports intrinsic size, so
  tall GIFs can't expand the layout.
- **Group avatars**: new `GroupAvatar` mosaic, used in DM rows + chat header
  when no explicit group picture is set.
- **Chat images**: tap to open in fullscreen lightbox (pinch-to-zoom,
  swipe-down to dismiss).
- **Chat GIFs**: render aspect-fit (no more zoom-crop), max 240×180.
- **Attachments picker**: rewritten as a UIKit `UICollectionView` +
  `PHCachingImageManager` + `PHPhotoLibraryChangeObserver` — fixes the glitch
  after granting access.
